Como os rótulos funcionam

  • Deixe o Intervalo de Páginas vazio para rotular todas as páginas.
  • Use estilos como números romanos para as páginas iniciais e numeração decimal para o corpo principal.
  • Use o Prefixo com Valor Inicial 0 para criar rótulos como A-0, A-1, A-2.
  • Ative o progresso quando uma regra usa um intervalo disjunto, como 1-9,30-40 ou ímpar.

Perguntas Frequentes

O que é um rótulo de página?

Um rótulo de página é a numeração exibida pelos leitores de PDF em miniaturas, navegação de páginas e caixas de diálogo de impressão. Ele não imprime um novo texto na própria página.

Posso misturar números romanos e números normais?

Sim. Adicione várias regras, como 1-4 com LowercaseRoman e 5-20 com DecimalArabic.

Meus arquivos permanecem privados?

Sim. O processamento acontece no seu navegador usando o CoherentPDF WASM. Seu PDF não sai do seu dispositivo.
